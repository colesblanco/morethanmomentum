⚡ Connect MTM's GHL Account
This tab pulls MTM's own GoHighLevel pipeline data — leads, revenue, and open opportunities — directly from the MCP Analytics Server.

Step 1 — Get MTM's GHL API Key and Location ID from GHL Settings → Integrations → API Keys.

Step 2 — Run this curl command to register MTM on the analytics server:
curl -X POST https://mtm-mcp-server.coleblanco.workers.dev/admin/clients \ -H "Content-Type: application/json" \ -H "X-Master-Key: YOUR_MASTER_KEY" \ -d '{ "name": "More Than Momentum", "ghlApiKey": "YOUR_MTM_GHL_API_KEY", "ghlLocationId": "YOUR_MTM_LOCATION_ID" }'
Step 3 — Copy the secret from the response.

Step 4 — In Cloudflare → your MTM Pages project → Settings → Environment Variables, add:
MTM_GHL_SECRET = the secret from Step 3.

Redeploy (git push), then reload this tab — the analytics dashboard will appear automatically.
